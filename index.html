About Me

I have been studying software development in my spare time since November 2015. I have also done some basic game development in Unity. Services that I offer

Web App/Site

Build responsive web sites and web applications that function and look great on small screens and large screens. I am able to create both the front end and back end/API, if needed. I will also help guide you through setting up domain names, hosting and any other services required.

Mobile App

Build a (Android and/or iOS) mobile application that will run on a variety of devices and screen sizes. I have a range of technologies to pick from and will work with you to choose the best and most inexpensive option for your project. I will also help you get it up and running on the required app stores, if needed.

Desktop App

Build a (Windows and/or Linux and/or MacOS) desktop application to your specifications. I use the cross-platform Electron framework to accomplish this so you are guaranteed the greatest compatability. I can also help you set up a website/server for distribution of your new app and handle updates, etc.
